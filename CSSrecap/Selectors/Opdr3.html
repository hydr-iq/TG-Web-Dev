Verschillende selectors en voorbeelden

  • Type selectors

    • De CSS type selector, ook wel bekend als tag name of element selector, matcht/selecteert een HTML tag/element van de desbetreffende type in een document. Voorbeeld:

    /* All elements. *

    a {

    color: red;

    }

    In het voorbeeld hieronder zie je dat specifieke paragraafelementen in HTML getagd zijn en in CSS gestyled zijn.

  • Descendant selectors

    • De descendant selector stelt je in staat om je te beperken tot elementen die afstammeling (descendant) zijn van een ander element. De syntaxis is erg eenvoudig, je geeft de naam van de parent (ouder), één of meer spaties, en dan de naam van het element dat je eigenlijke doel is. Hier volgt een voorbeeld:

    Hello, world - what a beautiful day!

    Hello, world - what a beautiful day!

    In dit voorbeeld willen we vetgedrukte tekst blauw maken, maar alleen als het behoort tot een alinea ("paragraph") gedefinieerd met (p) ... (/p). Je ziet in het voorbeeld dat er vier (b) tags voorkomen, de eerste twee binnen een paragraph element - ze zijn kinderen van (p) - en de laatste twee tags zijn niet ingesloten door paragraph tags. De inhouden van de eerste twee bold tags zijn blauw dankzij onze descendant selector. De inhouden van de laatste twee bold tags hebben de default kleur zwart.


  • Class selectors

    • De CSS class selector start met een punt(.). Het zal alles selecteren in het document waar deze class staat vermeld.

    In onderstaande bijvoorbeeld zijn specifieke delen van de tekst geel gearceerd door specifieke toepassing van tags in de HTML code op die delen van de tekst en vervolgens de styling ervan in de CSS sheet.
